# 🧑‍💼 Admin Panel (SaaS Dashboard)

A professional admin dashboard to manage users, products, orders, and analytics.

---

## 🚀 Features

* 🔐 Admin Authentication
* 📊 Dashboard Analytics (Revenue, Orders)
* 👥 User Management (CRUD)
* 📦 Product Management (CRUD)
* 🔄 Order Status Updates
* 🔍 Search & Pagination
* 🔔 Toast Notifications
* 👤 Profile Avatars

---

## 🛠️ Tech Stack

* React.js (Vite)
* Redux Toolkit
* Tailwind CSS
* Recharts (Charts)
* Axios

---

## 📁 Project Structure

```
src/
├── api/
├── app/
├── features/
├── components/
├── layouts/
├── pages/
└── utils/
```

---

## ⚙️ Environment Variables

Create `.env`:

```
VITE_API_URL=http://localhost:5000/api
```

---

## ▶️ Run Locally

```bash
npm install
npm run dev
```
