# 🛍️ E-commerce Frontend (User Panel)

A modern React-based frontend for users to browse products, manage cart, and complete purchases.

---

## 🚀 Features

* 🔐 User Authentication
* 🛍️ Product Listing & Details
* ❤️ Wishlist
* 🛒 Cart System
* 🚚 Shipping & Checkout
* 💳 Stripe Payment Integration
* 📱 Responsive UI

---

## 🛠️ Tech Stack

* React.js (Vite)
* Redux Toolkit
* React Router
* Axios
* Tailwind CSS

---

## 📁 Project Structure

```
src/
├── api/
├── app/
├── features/
├── components/
├── pages/
├── routes/
└── utils/
```

---

## ⚙️ Environment Variables

Create `.env`:

```
VITE_API_URL=http://localhost:5000/api
VITE_STRIPE_PUBLIC_KEY=your_public_key
```

---

## ▶️ Run Locally

```bash
npm install
npm run dev
```
